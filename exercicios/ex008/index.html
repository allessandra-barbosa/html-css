Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno. "Big" é uma tag obsoleto.

Texto deletado

Pode ser marcado um texto excluído para indicar que ele deve ser lido, mas não considerado. Utilizada a tag "del"

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Uilizada a tag "ins".

Texto sobrescrito

Para inserir coisas do tipo x20+3. Utilizada a tag "sup".

Texto subescrito

Para inserir coisas do tipo H2O. Utilizada a tag "sub".
